¿Qué es OpenGL?

¿Qué es OpenGL?

Desarrollado a principios de los 90, es una interfaz para comunicarse con hardware de graficación.

El GL de su nombre representa Graphics Library.

In a bold move, SGI cleaned up IRIS GL, removed all functionality that did not relate to computer graphics and released it to the public in 1992 as OpenGL (Open Graphics Library), a cross-platform standardized API for real-time computer graphics.

¿Qué es OpenGL?

Era lider en la industria de gráficos en tiempo real, definió estándares.

Principalmente por ser el único multiplataforma del momento

Aportaciones de OpenGL

Juegos de culto nacen con OpenGL.

Half Life, Quake 2 y Unreal mostraron todo el poder de OpenGL.

Capaz de aprovechar las nuevas GPU's de la época.

Junto a Direct3D, primeros en implementar el método de Buffer Objects.

Retrocompatibilidad y Multiplataforma.

Capaz de funcionar con versiones antiguas independientemnte del SO debido a su sistema de Context.

T&L moved the vertex transformation calculations and lighting calculations from the computer's CPU to the GPU. The main advantage of a GPU is that it does floating-point operations very quickly since the hardware is dedicated to this task, while a CPU specializes in integer and more general-purpose operations.


Buffers had been around for a while in the form of display lists and vertex arrays, but they each had their drawbacks. Display lists still used the immediate mode, and vertex arrays were stored in the system's memory so they had to be transferred to the GPU every single call.

¿Qué es GLUT?

¿Qué es GLUT?

Sus siglas representan Graphics Library Utility Toolkit.

Nos permite crear el contexto adecuado para desarrollar programas independientes del SO.

Estas tareas no forman parte de la especificación de OpenGL; por lo que, a su alrededor, diferentes bibliotecas (toolkits en la jerga de OpenGL) han ido apareciendo para dar solución en diferentes plataformas a la creación de ventanas y a la gestión de la entrada del usuario. En el sitio web de OpenGL [3], se mencionan tres: GLUT, Freeglut y GLFW

¿Qué es GLUT?

Es la libreria con la crearemos el contexto para OpenGl.

Permite gestionar las ventanas y manejar el input.

En nuestro caso se utilizará la distribución de FreeGlut.
