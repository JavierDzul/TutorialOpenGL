Interacción con OpenGL

En OpenGL, podemos capturar entradas del usuario, como el teclado y el mouse, para interactuar con la escena 3D.

  • Teclas para mover o rotar objetos.
  • El mouse para manipular la cámara.
  • Eventos de tiempo real para animación o interacción continua.

Captura de Entrada del Teclado

GLUT nos permite capturar eventos del teclado mediante la función glutKeyboardFunc:


void handleKeypress(unsigned char key, int x, int y) {
if (key == 'w') {
// Mover hacia adelante
}
if (key == 's') {
// Mover hacia atrás
}
glutPostRedisplay();
}

Se debe registrar la función de callback con glutKeyboardFunc(handleKeypress).

Manejo de Teclas Especiales

Además de las teclas normales, podemos manejar teclas especiales como las flechas de dirección con glutSpecialFunc:


void handleSpecialKeypress(int key, int x, int y) {
if (key == GLUT_KEY_UP) {
// Mover hacia arriba
}
if (key == GLUT_KEY_DOWN) {
// Mover hacia abajo
}
glutPostRedisplay();
}

Captura de Entrada del Mouse

GLUT también nos permite manejar eventos del mouse con glutMouseFunc:


void handleMouse(int button, int state, int x, int y) {
if (button == GLUT_LEFT_BUTTON && state == GLUT_DOWN) {
// Hacer algo cuando se hace clic con el botón izquierdo
}
}

Se puede registrar la función de callback con glutMouseFunc(handleMouse).

Captura de Movimiento del Mouse

Podemos capturar el movimiento del mouse cuando se arrastra con glutMotionFunc:


void handleMouseMotion(int x, int y) {
// Actualizar la vista o manipular objetos según el movimiento del mouse
glutPostRedisplay();
}

Implementación de Controles de Cámara

Utilizando la entrada del teclado y el mouse, podemos implementar un de control de cámara.

  • Teclas WASD para mover la cámara.
  • El mouse para rotar la vista.

void handleKeypress(unsigned char key, int x, int y) {
if (key == 'w') eyeZ -= 0.1f;  // Mover hacia adelante
if (key == 's') eyeZ += 0.1f;  // Mover hacia atrás
glutPostRedisplay();
}

void handleMouseMotion(int x, int y) {
// Cambiar la dirección de la cámara según el movimiento del mouse
glutPostRedisplay();
}

Ejercicio: Implementar Navegación en una Escena

Implementa un sistema de navegación en una escena 3D:

  • Utiliza las teclas WASD para mover la cámara en los ejes X, Y y Z.
  • Utiliza el mouse para rotar la vista de la cámara.

Resumen de lo aprendido

  • En OpenGL, podemos capturar entradas del teclado y mouse para interactuar con la escena.
  • Se pueden implementar controles personalizados para mover objetos o manipular la cámara.
  • La interacción es clave para hacer que la escena sea dinámica y reactiva al usuario.
