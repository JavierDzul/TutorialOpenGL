Instalación

De OpenGL y FreeGlut

Instalación de OpenGL

OpenGL ya está preinstalado en la mayoría de los sistemas, pero puedes verificar la versión en tu tarjeta gráfica.

  • Windows: Ejecuta dxdiag en la consola.
  • Alternativamente instala GlView

Instalación de Controladores
(De ser necesario)

Si es necesario, descarga los controladores más recientes desde el sitio web de tu fabricante.

Instalación de FreeGLUT en Windows

  1. Descargar desde FreeGLUT en SourceForge.
  2. Extraer el archivo descargado.
  3. Identificar los archivos:
    • freeglut.dll (Ejecutables)
    • freeglut.lib (Librerías)
    • freeglut.h (Cabeceras)

Configuración en Dev-C++

  1. Abrir Dev-C++ y crear un nuevo proyecto de C++.
  2. Seleccionar la opción Empty Project y elegir la plantilla **C++**.
  3. Agregar las rutas de los archivos de FreeGLUT:
    • Ve a "Project" -> "Project Options".
    • En la pestaña de Parameters, añade en la sección de "Linker" las siguientes bibliotecas:
      • -lglu32
      • -lopengl32
      • -lfreeglut

Copiar DLL a Directorio del Proyecto

Asegúrate de copiar el archivo freeglut.dll en la misma carpeta donde está el archivo ejecutable del proyecto (dentro de tu directorio de Debug o Release).

Código de Ejemplo Para Probar La Instalación: Crear Ventana


#include <GL/freeglut.h>

void display() {
glClear(GL_COLOR_BUFFER_BIT);
glFlush();
}

int main(int argc, char** argv) {
glutInit(&argc, argv);
glutCreateWindow("Ventana OpenGL");
glutDisplayFunc(display);
glutMainLoop();
return 0;
}

Compilación y Ejecución

Si has seguido correctamente los pasos, deberías poder compilar y ejecutar tu programa para ver una ventana básica de OpenGL.

Solución de Problemas Comunes

  • Error de Biblioteca no Encontrada: Verifica las rutas de las librerías en el proyecto.
  • Pantalla en Negro: Asegúrate de tener los controladores de gráficos actualizados.
  • Errores al Compilar: Revisa las rutas de los archivos de cabecera y las bibliotecas.
