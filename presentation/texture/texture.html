Texturizado

¿Qué es el Texturizado?

El texturizado es la técnica de aplicar una imagen (o "textura") sobre la superficie de un objeto 3D para darle detalle.

  • Las texturas permiten representar detalles sin necesidad de incrementar el número de polígonos.
  • Se mapean en las coordenadas del objeto utilizando coordenadas UV.

Coordenadas de Textura (UV)

Las coordenadas de textura o UV definen cómo una textura se ajusta a una superficie:

  • U: Representa la coordenada horizontal (ancho) de la textura.
  • V: Representa la coordenada vertical (alto) de la textura.

glTexCoord2f(u, v);

Se especifica antes de cada vértice para asignar la parte de la textura a ese vértice.

Carga de Texturas en OpenGL

Para cargar una textura en OpenGL, seguimos estos pasos:

  1. Cargar la imagen de la textura en memoria.
  2. Asignar la textura a un ID de OpenGL.
  3. Configurar parámetros de la textura (como filtrado y envoltura).

Código para Cargar una Textura


GLuint texture;
glGenTextures(1, &texture);  // Generar ID de textura
glBindTexture(GL_TEXTURE_2D, texture);  // Enlazar la textura

// Configuración de parámetros de la textura
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_REPEAT);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_REPEAT);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);

Filtros de Textura

Los filtros de textura definen cómo se mapea una textura cuando es escalada o minimizada:

  • GL_LINEAR: Filtrado lineal, suaviza la textura cuando se escala.
  • GL_NEAREST: Filtrado cercano, utiliza el píxel más cercano (da un aspecto pixelado).

Aplicación de Texturas a un Objeto

Una vez cargada la textura, se debe mapear a un objeto:


glEnable(GL_TEXTURE_2D);  // Habilitar texturas 2D
glBindTexture(GL_TEXTURE_2D, texture);

glBegin(GL_QUADS);  // Dibujar un cuadrado con textura
glTexCoord2f(0.0f, 0.0f); glVertex3f(-0.5f, -0.5f, 0.0f);  // Esquina inferior izquierda
glTexCoord2f(1.0f, 0.0f); glVertex3f( 0.5f, -0.5f, 0.0f);  // Esquina inferior derecha
glTexCoord2f(1.0f, 1.0f); glVertex3f( 0.5f,  0.5f, 0.0f);  // Esquina superior derecha
glTexCoord2f(0.0f, 1.0f); glVertex3f(-0.5f,  0.5f, 0.0f);  // Esquina superior izquierda
glEnd();

Configuración de la Envoltura de Textura

La envoltura de textura define cómo se comporta la textura cuando las coordenadas UV están fuera del rango [0,1]:

  • GL_REPEAT: Repite la textura.
  • GL_CLAMP_TO_EDGE: Extiende el último píxel en los bordes.

glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_REPEAT);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_REPEAT);

Ejercicio: Aplicar Texturas a un Cubo

Implementa texturas en las caras de un cubo:

  • Carga una textura desde un archivo de imagen.
  • Asigna coordenadas UV a cada vértice del cubo.
  • Configura el filtrado y la envoltura de la textura.

Resumen de lo aprendido

  • El texturizado permite aplicar imágenes sobre superficies 3D.
  • Las coordenadas UV definen cómo una textura se ajusta a una superficie.
  • OpenGL permite configurar los filtros de textura y la envoltura para controlar su comportamiento.
